# Plan de Estudio: Django Esencial

Plan de estudio para aprender Django desde cero hasta producción. Enfocado en conceptos esenciales, recomendaciones prácticas y lo que se usa más en el mundo real.

## Fase 1: Fundamentos (1-2 semanas)
- [ ] Instalación y Setup: Python, virtualenv, Django. Crear proyecto y app.
- [ ] Modelo-Vista-Template (MVT): Entender patrón Django.
- [ ] Modelos y Migraciones: Crear modelos, makemigrations/migrate.
- [ ] URLs y Vistas Function-Based: Configurar rutas básicas.

**Recomendaciones:** Usar SQLite. Leer "Django for Beginners".

## Fase 2: CRUD y Forms (1 semana)
- [ ] Forms y Validación: ModelForm, validación.
- [ ] Admin Interface: Registrar modelos, customizar.
- [ ] Vistas Class-Based (CBV): ListView, DetailView.
- [ ] Templates: Tags, filters, inheritance.

**Recomendaciones:** Proyecto CRUD pequeño. Bootstrap para estilos.

## Fase 3: BD Avanzada y Seguridad (1 semana)
- [ ] Querysets y ORM: Filtros, agregaciones.
- [ ] Autenticación: User, login/logout, permisos.
- [ ] Static Files y Media: Configurar archivos.
- [ ] Middleware y Seguridad: CSRF, DEBUG=False.

**Recomendaciones:** PostgreSQL en prod. Django Allauth para social login.

## Fase 4: Deployment y Producción (1 semana)
- [ ] Settings por Entorno: SECRET_KEY, ALLOWED_HOSTS.
- [ ] Deployment: Heroku, Railway.
- [ ] Testing: Unit tests.
- [ ] Extras: Signals, Django Debug Toolbar.

**Recomendaciones:** Deploy temprano. Comunidad: r/django.

**Consejo:** Escribir código vos mismo. Duración: 2-3 meses.
