# Proyecto de Aprendizaje: Django y Django REST Framework

Este repositorio contiene una serie de unidades de aprendizaje y ejemplos prácticos centrados en Django y Django REST Framework (DRF).

## Estructura

-   **/docs**: Documentación principal del proyecto.
    -   [Fundamentos de Django](README_Django.md)
    -   [Fundamentos de DRF](README_DRF.md)
-   **/learning_units**: Lecciones específicas organizadas por tema.
    -   [Unidad 1: DRF Intermedio](./../learning_units/unidad_1_drf_intermedio/README.md)
-   **/src**: Código fuente del proyecto de ejemplo (una aplicación de tickets).

## Propósito

El objetivo es servir como un recurso práctico y educativo, donde cada "unidad de aprendizaje" explora un concepto específico a través de explicaciones y código de ejemplo.
